條件判斷在python裡很常會用到。
最基本的是if，如果在if後面的條件判斷成立的話，就會執行冒號後的內容，範例如下:
```python
score = int(60)
if score >= 60:
	print("及格")

#輸出
及格
```
要注意的是，python跟C++不同，不是使用{}而是使用縮排(Tab鍵，預設是4個空格)。
```Cpp
#include <iostream>
using namespace std;

int sorce = 60;

int main()
{
    if (sorce >= 60)
    {
        cout << "及格";
    }
    return 0;
}

//輸出
//及格
```
if(如果)還可以搭配else(否則)跟elif(又或者)來使用
