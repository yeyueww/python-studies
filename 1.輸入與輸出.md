### 輸出文字 #
```python
print("Hello World")
print("Hello python")

#輸出
Hello World
Hello python
```
預設輸出後會換行，若不想換行可寫成以下:
```python
print("Hello World",end="")
print("Hello python")

#輸出
Hello WorldHello python
```
### 輸入文字
```python
input()
```
如果想在輸入時會有字顯示的話:
```python
input("Your input:")

#輸出
Your input:
```
